# 📊 Mobile Sales Analytics Dashboard

## 📌 Project Overview

The **Mobile Sales Analytics Dashboard** is an interactive Power BI report designed to analyze and visualize mobile phone sales performance across different brands, cities, months, and payment methods.

This dashboard provides meaningful business insights such as total sales, total quantity sold, transaction trends, customer ratings, and payment distribution to support data-driven decision-making.

---

## 🎯 Objectives

* Analyze overall mobile sales performance
* Track monthly sales and quantity trends
* Compare brand-wise performance
* Monitor customer ratings distribution
* Analyze transactions by payment method
* Identify best-performing days and models

---

## 🛠 Tools & Technologies Used

* **Power BI** – Data visualization & dashboard development
* **Excel / CSV Dataset** – Data source
* **DAX (Data Analysis Expressions)** – Calculated measures
* **Data Cleaning & Transformation** – Power Query


## 📊 Dashboard Features

### 🔹 Sales Analysis

* Total Sales by City
* Total Sales by Mobile Model
* Total Sales by Day Name
* Monthly Quantity Trend

### 🔹 Brand Performance

* Brand-wise Sales
* Total Quantity by Brand
* Transaction Count by Brand

### 🔹 Customer Insights

* Customer Ratings Distribution
* Transactions by Payment Method (UPI, Credit Card, Debit Card, Cash)

### 🔹 Interactive Filters

* Mobile Model
* Payment Method
* Brand
* Day Name
* Month Selection

---



## 🚀 How to Use

1. Download the `.pbix` file

3. Interact with filters and visuals
4. Explore insights from different perspectives
